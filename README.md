Conception de Base de Données - Salle de Sport

Ce projet présente la modélisation et la gestion d'une base de données pour une salle de sport.

Contenu du projet :
Modélisation : Diagramme Entité-Association (MCD/EA) pour structurer les données.

SQL : Requêtes de création et de manipulation de données (salles, utilisateurs, abonnements).

C# : Interface de gestion des sessions et connexion à la base de données.
